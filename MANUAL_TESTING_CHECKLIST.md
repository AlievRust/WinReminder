## Тестовые сценарии

### 1. Запуск приложения

**Шаги:**
1. Запустить приложение: `python src/main.py`
2. Проверить, что открывается главное окно с заголовком "Reminder"
3. Проверить размер окна (900x600)
4. Проверить наличие всех элементов интерфейса:
   - Кнопки: Добавить, Готово, Отменить, Удалить, Тест уведомления
   - Фильтр по статусу (combobox)
   - Список напоминаний (Treeview)
   - Статусная строка внизу

**Ожидаемый результат:**
✅ Приложение запускается без ошибок
✅ Все элементы интерфейса отображаются корректно
✅ Статусная строка показывает "Всего напоминаний: 0"

### 2. Добавление напоминания

**Шаги:**
1. Нажать кнопку "Добавить"
2. Проверить, что открывается диалог размером 550x680
3. Заполнить поля:
   - Заголовок: "Тестовое напоминание"
   - Описание: "Это тестовое описание"
   - Дата и время: использовать календарь и спинбоксы
4. Нажать кнопку "Сохранить"

**Ожидаемый результат:**
✅ Диалог открывается с фиксированным размером
✅ Все элементы видны без скроллирования
✅ Напоминание появляется в списке
✅ Статус: "Ожидает"
✅ Повторение: "Нет"
