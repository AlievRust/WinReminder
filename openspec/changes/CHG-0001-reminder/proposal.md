# Proposal: Reminder Application (change-001)

## Обзор
Создание мини-приложения Reminder для Windows 10/11 с графическим интерфейсом на базе Tkinter. Приложение позволит пользователям создавать, просматривать и управлять напоминаниями с автоматическими уведомлениями.

## Цель
- Обеспечить удобный инструмент для управления напоминаниями
- Показать уведомления поверх всех окон при наступлении времени
- Поддерживать повторяющиеся напоминания
- Быстрый MVP с возможностью масштабирования

## Критерии успеха
1. Приложение запускается на Windows 10/11
2. Уведомления появляются поверх всех окон при свёрнутом приложении
3. Напоминания сохраняются между сессиями
4. Статус "Просрочено" проставляется автоматически

## Выбранный стек
**Tkinter + plyer + threading + APScheduler**

Обоснование:
- ✅ Tkinter в стандартной библиотеке - быстрый старт
- ✅ plyer для системных уведомлений Windows
- ✅ threading/APSсheduler для фоновой проверки времени
- ✅ Только Win10/11 - упрощает тестирование

## Функциональные требования

### Основной функционал
- [ ] Добавление напоминаний (заголовок, описание, дата/время)
- [ ] Удаление напоминаний
- [ ] Просмотр всех напоминаний с фильтрацией по статусу
- [ ] Изменение статуса: "Готово", "Отменено"
- [ ] Автоматическая установка статуса "Просрочено"

### Уведомления
- [ ] Всплывающие уведомления поверх всех окон
- [ ] Работа уведомлений при свёрнутом приложении
- [ ] Кнопка "Тест уведомления"

### Повторяющиеся напоминания
- [ ] Повторение: час, день, неделя, месяц

### Быстрая настройка времени
- [ ] Кнопки: "через 1 мин", "через 15 мин", "через 30 мин", "через 1 час"

### Статусы напоминаний
- "Ожидает" - новое или ожидающее срабатывания
- "Готово" - выполнено пользователем
- "Просрочено" - время прошло, не выполнено
- "Отменено" - отменено пользователем

## Нефункциональные требования
1. **Производительность:** проверка напоминаний не должна блокировать GUI
2. **Надёжность:** данные не должны теряться при аварийном закрытии
3. **Совместимость:** Windows 10/11, Python 3.10+
4. **UX:** звук уведомлений согласно настройкам ОС

## Риски и митигации
| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| plyer не работает на Win11 | Средняя | Высокая | Резерв: win10toast |
| Threading блокирует GUI | Низкая | Средняя | Использовать APScheduler |
| Блокировка БД при многопоточности | Средняя | Высокая | Connection per thread или lock |
