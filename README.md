# Reminder Application (educational project)
**vibe-coding, GLM-4.7, spec-driven development (OpenSpec)**

Мини-приложение для управления напоминаниями на Windows 10/11 с автоматическими уведомлениями.

## Возможности

- ✅ Добавление, удаление и просмотр напоминаний
- ✅ Управление статусами: "Ожидает", "Готово", "Просрочено", "Отменено"
- ✅ Повторяющиеся напоминания: час, день, неделя, месяц
- ✅ Быстрая настройка времени: +1 мин, +15 мин, +30 мин, +1 час, +3 часа, +1 день
- ✅ Системные уведомления (работают при свёрнутом приложении)
- ✅ Фильтрация напоминаний по статусу
- ✅ Автоматическое проставление статуса "Просрочено"
- ✅ Сохранение данных между сессиями (SQLite)

## Требования

- Python 3.10 или выше
- Windows 10/11

## Использование

### Запуск приложения

```bash
python src/main.py
```

### Добавление напоминания

1. Нажмите кнопку "Добавить"
2. Введите заголовок и описание (опционально)
3. Выберите дату в календаре и время с помощью спинбоксов
4. Используйте быстрые кнопки (+1 мин, +15 мин и т.д.) для быстрого выбора времени
5. При необходимости выберите интервал повторения
6. Нажмите "Сохранить"

**Примечание:** Диалог добавления имеет фиксированный размер, все элементы гарантированно видны без скроллирования.

### Управление напоминаниями

- **Готово**: отметить выбранное напоминание как выполненное
- **Отменить**: отменить выбранное напоминание
- **Удалить**: удалить выбранное напоминание
- **Фильтр**: отфильтровать список по статусу

### Тестирование уведомлений

Нажмите кнопку "Тест уведомления" для проверки работы системных уведомлений.
При успешной отправке всплывающее сообщение не появляется (оно отображается в системном центре уведомлений).

## Запуск тестов

```bash
# Запуск всех тестов
pytest

# Запуск с выводом деталей
pytest -v

# Запуск только тестов базы данных
pytest tests/test_database.py -v

# Запуск только тестов модели
pytest tests/test_reminder.py -v

# Запуск только тестов планировщика
pytest tests/test_scheduler.py -v
```
